# 3. ĐỊNH NGHĨA & THUẬT NGỮ

| **<span style="color: inherit; background-color: rgba(186,206,253,0.7)">Thuật ngữ</span>** | **<span style="color: inherit; background-color: rgba(186,206,253,0.7)">Định nghĩa</span>** |
| ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------- |
| **CR (Conversion Rate)**                                                                   | Tỷ lệ chuyển đổi = Số đơn thành công / Số lượt tiếp cận   (hoặc tư vấn) × 100%              |
| **Bill Size**                                                                              | Giá trị trung bình mỗi đơn hàng/hóa đơn = Tổng doanh thu /   Số đơn                         |
| **ROI**                                                                                    | Return on Investment = (Doanh thu - Chi phí) / Chi phí ×   100%                             |
| **LNG/LNG%**                                                                               | Lợi nhuận gộp / Tỷ lệ lợi nhuận gộp trên doanh thu                                          |
| **Redemption Rate**                                                                        | Tỷ lệ sử dụng voucher = Số voucher đã dùng / Số voucher   phát hành × 100%                  |
| **Kill Switch**                                                                            | Ngưỡng dừng khẩn cấp - khi chỉ số xuống dưới mức này phải dừng chương trình ngay            |
